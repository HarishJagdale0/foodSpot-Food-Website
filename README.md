### Run Locally

To run **Foodie** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/HarishJagdale0/foodSpot.git
```

Windows:

```bash
git clone https://github.com/HarishJagale0/foodSpot.git
```
